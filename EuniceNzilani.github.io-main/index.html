Recent Work & Achievements

Data Visualization Dashboard: VisualAI

VisualAI is an interactive data visualization platform that transforms complex datasets into intuitive, explorable visualizations. Inspired by Tableau's functionality but built from the ground up, this tool empowers users to derive meaningful insights through customizable dashboards.

The application leverages React for the frontend with D3.js/Recharts for dynamic visualizations, while a Node.js backend handles data processing. Users can upload Excel spreadsheets and instantly generate professional charts, graphs, and maps with AI-assisted recommendations for optimal visualization types based on data patterns.

VisualAI features real-time data updates, customizable views, advanced filtering capabilities, and export options—making data exploration accessible to both analysts and non-technical users alike.
